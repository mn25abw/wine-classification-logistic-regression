Logistic Regression for Wine Classification

Project Overview

This project demonstrates the implementation of Logistic Regression for a multi-class classification problem using the Wine dataset. It focuses on both model performance and interpretability.

 Dataset
Dataset: Wine Dataset (from sklearn)
Total Samples: 178
Features: 13 chemical properties
Classes: 3 wine categories

 Technologies Used
Python
Jupyter Notebook
Libraries:
NumPy
Pandas
Matplotlib
Seaborn
Scikit-learn

 Project Workflow
Data Loading
Exploratory Data Analysis (EDA)
Feature Scaling
Model Training (Logistic Regression)
Model Evaluation
Interpretation of Coefficients
 Results
Achieved ~97% accuracy
Very low misclassification
Strong feature importance insights
Features like Alcohol, Proline, and Color Intensity play a major role
Some multicollinearity observed
Logistic Regression performs well on structured datasets
